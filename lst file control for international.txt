
※ 국가별로 아이템 및 퀘스트, 캐쉬샵에서 판매하는 품목의 변화가 필요하여
   다음 스크립트들에 대해서 국가별로 추가스크립트가 지원되게 되었습니다.

	대상 스크립트 :
		Equipment/equipment.lst
		Stackable/stackable.lst
		Quest/quest.lst
		Etc/NewCashshop.etc


   이것들 말고도 앞으로 필요에 따라 추가될 예정입니다.







※ 이를 위하여 공통lst파일을 하나 두고, 그외 국가별로 다른 사항들만 모아두는
   국가별lst 를 두기로 했습니다.
   각 국가별 추가 lst 는 맨 끝에 다음의 확장자가 추가됩니다.

	공통	없음
	한국	.kor
	일본	.jap
	중국	.chi

 그래서 이렇게 됩니다.
	Equipment/equipment.lst		// 공통 장비들
	Equipment/equipment.lst.jap	// 일본에만 들어가는 장비들
	Equipment/equipment.lst.kor	// 한국에만 들어가는 장비들
	Equipment/equipment.lst.chi	// 중국에만 들어가는 장비들

 각 국가별로 자신에 해당하는 lst를 추가적으로 로드하게 됩니다.






※ 주의사항

Equipment/equipment.lst
Stackable/stackable.lst
Quest/quest.lst
위 파일들의 경우 번호가 중복되면 안됩니다.

즉 

Equipment/equipment.lst.jap 에 1000번 기모노 장비가 있었는데
Equipment/equipment.lst.kor 에 1000번 한복 장비가 있다면

당연히 에러입니다!!



Etc/NewCashshop.etc 의 경우 2가지가 더 추가됩니다.

첫째는 
Etc/NewCashshop.etc.jap 에 IPG번호 3000번, 아이템번호 555번 코인 30개 셋트
Etc/NewCashshop.etc.kor 에 IPG번호 3000번, 아이템번호 555번 코인 10개 셋트

이렇게 IPG번호 도 중복되면 안되고 아이템번호도 당연히 중복되면 안됩니다.
( 특히 호준씨 주의 - 캐쉬샵에서 버그나면 성과급이 없어질지도!!! ㅜㅁㅜ)/ )


둘째는
공통 Etc/NewCashshop.etc 과
국가별 Etc/NewCashshop.etc.jap 의 태그가 다릅니다.
자세한 내용은 Etc/NewCashshop.etc.jap 에 나와있습니다.
